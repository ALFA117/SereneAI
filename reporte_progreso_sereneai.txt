2. INSTRUCCIONES DE LA IA — SISTEMA  ✅ COMPLETO (NUEVO)
────────────────────────────────────────────────────────────────────────────────

  Archivo: backend/sereneai_system_prompt.py

  Funciones implementadas:
  ┌─────────────────────────────────┬──────────────────────────────────────────┐
  │ Función                         │ Descripción                              │
  ├─────────────────────────────────┼──────────────────────────────────────────┤
  │ detectar_crisis(texto)          │ Detecta 25+ keywords de crisis/suicidio  │
  │                                 │ ANTES de llamar al LLM                   │
  ├─────────────────────────────────┼──────────────────────────────────────────┤
  │ limpiar_respuesta(texto)        │ Elimina bloques <think>...</think>        │
  │                                 │ que DeepSeek R1 añade antes de responder │
  ├─────────────────────────────────┼──────────────────────────────────────────┤
  │ build_system_prompt(contexto)   │ Construye el system prompt completo con: │
  │                                 │ - Personalidad empática de SereneAI      │
  │                                 │ - Reglas éticas (sin diagnóstico)        │
  │                                 │ - Contexto RAG de las 3 técnicas top     │
  │                                 │ - Formato de respuesta (sin markdown)    │
  ├─────────────────────────────────┼──────────────────────────────────────────┤
  │ build_image_prompt(frase)       │ Genera el prompt para Oxlo Image Pro     │
  │                                 │ basado en la frase de cierre del LLM     │
  ├─────────────────────────────────┼──────────────────────────────────────────┤
  │ PROMPT_FRASE_CIERRE             │ Prompt para que el LLM genere la frase   │
  │                                 │ corta de la tarjeta motivacional         │
  ├─────────────────────────────────┼──────────────────────────────────────────┤
  │ MENSAJE_CRISIS                  │ Dict fijo con texto empático + 3 líneas  │
  │                                 │ de ayuda (SAPTEL, Línea de la Vida, IMSS)│
  └─────────────────────────────────┴──────────────────────────────────────────┘

  Reglas del system prompt:
  ✅ Tono cálido, en español, sin tecnicismos
  ✅ Respuestas de 80-200 palabras (evita muros de texto)
  ✅ Sin markdown, sin bullets (el LLM habla como persona)
  ✅ Solo una técnica por respuesta
  ✅ Siempre termina con una pregunta abierta
  ✅ No menciona que es IA a menos que se le pregunte
  ✅ No diagnostica, no recomienda medicamentos
